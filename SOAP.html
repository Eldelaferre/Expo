    Que es el SOAP?

    El Protocolo simple de acceso a objetos (SOAP) es un protocolo ligero basado en XML para intercambiar información en un entorno distribuido y descentralizado. Al combinar solicitudes y respuestas basadas en SOAP con un protocolo de transporte, como HTTP, Internet se convierte en un medio para que las aplicaciones publiquen servicios web respaldados por bases de datos

    SOAP tiene un acoplamiento más flexible entre el cliente y el servidor que algunos protocolos de computación distribuida similares, como CORBA / IIOP, y proporciona una comunicación más fácil para un cliente y un servidor que usan diferentes lenguajes. SOAP expone una forma estándar para que los procesos se comuniquen, pero aprovecha las tecnologías existentes.

    Las solicitudes SOAP son fáciles de generar y un cliente puede procesar fácilmente las respuestas. Una aplicación puede convertirse en un cliente programático de los servicios de otra aplicación, y cada una de ellas intercambia información rica y estructurada. La capacidad de agregar servicios web potentes y distribuidos permite que SOAP proporcione un modelo de programación sólido que convierte Internet en una plataforma de desarrollo de aplicaciones.
